The Ruby Enumerable Method "Cycle"

Enumerables

Ruby's "classes" can reference "mixins" such as "enumerable." A ruby class is like a blueprint for a certain virtual item in a computer's memory. If the virtual item we were trying to create was a bluejay (the bird), we might use the imaginary class "Bird". We could also use Bird to create a finch or an eagle. The bluejay, finch, and eagle would all be instances (objects) of the class Bird. If that class included the mixin, "flight", then we would be able to use the things that are included in the "flight" collection. In this way, Ruby's classes, such as "fixnums" (which are just numbers such as 1, 2, and 3) include some mixins, including "enumerable".

For a class to use enumerable, the objects of the class (the things created by the class and that belong to the class) must be accessible via #Each. The string of characters, "abcdef" is not accessible with each:

"abcdef".each #=> NoMethodError: undefined method 'each' for "abcdef":String

However it is accessible via each_char:

"abcdef".each_char

The range of numbers 1-3 is accessible via each:

(1..3).each {|i| print i} #=>1;2;3;

Enumerable also allows the object of a class that includes it to be searched, sorted, and selected with methods like :find, :sort, and :select.

Cycle

Cycle is an enumerable method that runs a block, a set of instructions, a number of times, or if no number of times is given, it runs the block infinitely.

(1..3).cycle(3) { |i| puts i } #=>1, 2, 3, 1, 2, 3, 1, 2, 3

In that example, we used :cycle on the range "1, 2, 3", shown as (1..3). We told cycle to run the block three times .cycle(3). The block was the section in curly brackets, { |i| puts i} and follows the form, for the object represent by the variable "i" (it can be called anything, I picked "i"), puts i, which means print the string version of i. In this case the object represented by the variable on cycle's first run of three, is the first number in the range 1-3, followed by the second number, and lastly the third number. So cycle prints 1, 2, 3 and then starts the second cycle1, 2, 3. :Cycle can be used with any block, not just puts i!
