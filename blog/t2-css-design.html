When to Use Classes or IDs in HTML and CSS

Global Styling

An easy way to use CSS to style an element, such as making the font blue in your main body of text would be,
HTML: 

<p>Hi, my name is Adell</p>

CSS: 
p {
    color: blue;
}
This is called global styling and is usually not a good practice since it will apply the styling to all of your text within <p>s.

IDs

IDs are a good way to identify styling for one element on a page, but as a best practice should not be used for recurring elements. Use them like so:
HTML: 
<p id="introduction">Hi, my name is Adell!</p>

CSS: 
#introduction {
    color: blue;
}

Classes

Classes should be used whenever you're identifying the styling for a recurring element on the page. If you have, for example, multiple paragraphs, each denoted by a <p>, you should use classes.
HTML: 
<p class="mainBody">Hi, my name is Adell!</p>
<p class="mainBody">I like to read.</p>

CSS: 
p.mainBody {
    color: blue;
}
