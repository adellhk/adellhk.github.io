Arrays and Hashes

Arrays

Arrays are groups of values. Each value in an array 

my_array = ["I", "am", "Adell"]

can be accessed by its numerical location, called an index. The index count starts at 0. We can access the first element of my_array, "I", with 
my_array[0]

and the element "Adell" with 
my_array[2]

We can also access the first and last elements in an array with the methods
my_array.first

or
my_array.last

Arrays can hold any kind of object, such as a string, integer, or even another array.

Hashes

Hashes are like arrays, except instead of using a numerical index to access the values, we use keys. All of the values in a hash are stored with a corresponding key. These are called key-value pairs. An example hash would be

my_hash = { "key_1" => "I", "key_2" => "am", "key_3" => "Adell" }

To access the value "Adell", we would do
hash[key_3]

We can also store multiple elements at a key location in a hash by putting them in an array, such as 
my_hash_2 = { "key_x" => "I", "key_array" => ["am", "Adell"]}

Like arrays, hashes can store any kind of object as a value. Unlike arrays, their keys can also be any kind of object. We can use an integer, string, or symbol as a key, such as
my_hash_3 = { 1 => "I", :2 => "am", "third_key" => "Adell" }

When to Use Hashes or Arrays

Generally, we use hashes when we need to store a value with a specific key. Naturally, if we need to store a value with a key that is not an integer, we must use a hash, but it is also better to use a hash when we are using integers as keys, unless they start very close to zero and increment by ones, otherwise we will have a lot of wasted space. For example if you have a product cataloguing system and wish to pair product IDs to the products "Greatest Hits CD", "Plastic Water Bottle", and "Phone Case", you could do this efficiently with an array if you were content with the product IDs being 0, 1, and 2.

product_catalogue = ["Greatest Hits CD", "Plastic Water Bottle", "Phone Case"]

You could also do this moderately efficiently with an array if you wanted your product IDs to be 1, 5, and 8.
product_catalogue = [nil, "Greatest Hits CD", nil, nil, nil, "Plastic Water Bottle", nil, nil, "Phone Case"]

However if you wanted your product IDs to be 102934, 183924, and 394102, you would need to us a hash.
product_catalogue = { 102934 => "Greatest Hits CD", 183924 => "Plastic Water Bottle", 394102 => "Phone Case"}
